# SupplyChain Optimizer

**Otimização de rotas logísticas em São Paulo com Reinforcement Learning (Q-Learning)**

## Funcionalidades
- Roteamento otimizado de 20 pontos
- Custo mínimo com base em distância + demanda
- Mapa interativo com Folium
- Interface com Gradio

## Tech Stack
- Python, Gradio, Folium, Geopy, Q-Learning
